Windows环境优化

1，关闭任务栏用不到的内容

2，关闭自动更新

a, 先关闭自动更新的服务

在任务栏右键 ——> 打开“任务管理器” ——> 点击“服务” ——> 找到右上角的“打开服务”（如下图）

点击“Windows 更新”或者”Windows Update“，点击”停止“

点击”恢复“ ——> 第一次失败后面的：无操作”，然后点应用

只关了这里还不行，它还是会自动启动。

b，改组策略

Windows +Ｒ打开运行，输入 gpedit.msc

打开“本地组策略编辑器” ——> “用户配置” ——> “管理模板” ——> “系统” ——> Windows 自动更新，勾选“已启用”（如下图）

c，关闭安全相关的内容

打开“设置” ——> “隐私和安全性” ——> “Windows 安全中心”

打开 Windows 安全中心

打开“病毒和威胁防护”

点击“管理设置”

打开“排除项”，将 C盘和 D 盘都添加进去。

然后全部关掉

最后关闭防火墙。
