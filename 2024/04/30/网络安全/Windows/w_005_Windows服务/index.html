2，Windows服务

2.1 概念

Microsoft Windows服务（即，以前的NT服务）使你能够创建在它们自己的Windows会话中可长时间运行的可执行应用程序，这些服务可以在计算机启动时自动启动，可以暂停和重新启动而且不显示任何用户界面。这种服务非常适合在服务器上使用，或任何时候，为了不影响在同一台计算机上工作的其它用户，需要长时间运行功能时使用。

Windows服务是一种运行于Windows操作系统之上的一种独立进程。它们不必依赖于任何界面，它们由Windows操作系统本身负责管理维护，可以持续运行，不需要用户的参与，也可以接受来自外部的请求，一般来运行一些持续运行的后台进程，产生一些定期调度任务或者监测一些外部事件。

2.3 常见服务

DHCP Client 服务

这个服务程序如果没有开启，那么我们的电脑无法得到IP地址，就无法去联网。

Print Spooler 服务

没有这个服务程序，我们就没有办法打印东西。

VMware Tools服务

这是安装Vmware Tools工具软件的时候，Vmware自动在系统中添加的服务程序，就是为了一开机，虚拟机就能和物理主机进行数据交互等功能。

Windward Update服务

电脑能够自动检测到需要更新的原因，就是这个程序一直在后台默默运行着，检测并提示系统更新。

软件将自己添加到系统服务中的好处是，可以让这个程序开机自启动，每次开机直接就为用户提供服务，用户不需要自己去开启，不需要自己去考虑这个程序的运行，它就能默默的提供服务。
