Windows 基础操作

dll 文件

dll 文件是应用程序扩展文件

dll文件，学名是动态链接库（Dynamic Link library），DLL是一个包含可由多个程序，同时使用的代码和数据的库。

通过dll文件封闭一些公用的程序代码，其它程序想用这个功能，直接加载这个 dll 链接库文件即可。

图标文件

桌面快捷方式等大多数软件自带的图标效果，里面的图标文件是 .ico 结尾。

查看图标文件和所在位置如下：

如果想换自己设定的图标的话，就需要自己去程序目录中去找一下这个程序的图标在哪里，图标文件的名称是什么，然后自己去网上下载一些自己想用的图标，或者自己设计一个，再改变同名的图标，然后替换到程序的图标文件位置即可。

重要文件夹介绍

C盘下重要文件夹

  • Program Files 软件程序存放目录，64位程序文件默认会安装到这个目录下，安装的时候是可以自行修改安装目录，如果安装某个软件时，没有特意修改安装路径，就会被安装到这个目录中。
  • Program Files（x86） 32位程序文件默认会安装到这个目录下，x86是 x86_32的简写，x64 是 x86_64 的简写
  • Windows 系统程序的核心目录
  • 用户 多用户文件（桌面），现在的系统都支持多个用户登录操作，系统为了作区分，就在系统盘为每个用户分配一个默认文件夹，也就是用户目录

文件共享

文件共享，将某些文件分享给其它人。

1，网络工具

小文件传输可以直接用qq，微信等，大一些的文件可以用网盘，比如百度网盘等。但是百度网盘都有上传文件大小限制，比如百度网盘限制单个文件大小为30G，那么如果有一个超大的文件，大于30G，怎么办？

1.1 大文件压缩为分块小文件

如果压缩文件大小超过了30G，那么就可以在压缩文件的时候选择分块压缩，将大文件压缩成几个小文件。

效果如下

每种压缩软件压缩的文件样式不一样，有的是很多同名的，后缀就是01.rar.02.rar等，选中其中任意一个，然后解压即可得到完整的文件。

2，物理工具

如果双方距离比较近，直接用u盘，移动硬盘等。

4,远程控制/远程桌面

当你还在电脑所在地，又需要控制这台电脑时，就老需要用到远程控制。

远程桌面是系统自带的功能，也有一些第三方工具，比如向日葵比较好用，teamviewer也还可以，使用频繁会提示收费，QQ的远程比较拉跨，比较慢，todesk也不错，玩游戏的人喜欢用parser，挺好用。这些工具圾一个共同的特点，需要第三方服务器中转一下，所以需要网络比较流畅才行。
