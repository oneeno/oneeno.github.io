Linux Shell

1、Shell 的意思

Shell：蛋壳的意思，是Linux中比较重要的一个概念，所有的命令其实都称之为shell命令。

看图解：Shell 就是内核的一个外壳，用户通过Shell 来控制内核进而驱动硬件做事情，这是它名字的由来，Linux下，没有Shell的话，就不能控制这个计算机了，因为内核是用户不能直接控制的。
