Linux输入输出重定向

输入输出只能对文件进行操作，不能对目录操作

输出

>> 输出追加重定向，不会清空原文件

标准正确输出重定向1，标准错误输出重定向2

输出重定向用得比较多，输入重定向用得很少。下面是输入重定向简单的演示

输入

< 输入重定向

<< 输入追加重定向
