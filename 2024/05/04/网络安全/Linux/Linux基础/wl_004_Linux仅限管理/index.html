root 用户权限最高，其它用户需要设定权限并遵守权限

使用 lisi 来登录，并创建文件，将文件权限全部去掉，修改权限用 chmod 指令，全称 change mode

现在这个文件是没有任何权限的，但是文件是lisi创建的，文件属主还是lisi，虽然显示 lisi也没有权限，但是实际上 lisi是可以修改文件权限的，其它用户除了root用户，是没有权力修改这个文件权限的。

wangwu 用户

写权限的作用

wangwu用户想编辑文件，但是没有编辑权限，可以用 vi 打开，但是编辑之后不能保存

可执行权限的作用

需要创建一个命令文件才能看到效果，可以复制某个命令文件过来，谁复制过来的，这个文件的属主就是谁

切换到 wangwu 来执行一下 ls 这个文件

注意：不能直接 ls，直接 ls 还是调用系统 /bin/ls 文件，需要写 ./ls 才是使用当前目录下的 ls 文件，或者写这个文件的绝对路径 /tmp/ls，或者将它放到某个特定目录下，就可以直接使用对应指令而不用管路径了

可执行程序特殊目录说明

通过 echo $PATH 可以看到，类似于Windows环境变量中的PATH，反式放到这个目录中的命令程序，

还可以在任意目录下通过这个命令程序名称来直接调用命令执行。
