vi 编辑内容原理

注意：如果内存中的数据没有修改，不影响。如果修改了，并且不小心断开链接，也就是没有正常退出vi，那么隐藏文件会一直在硬盘上，当我们再次vi修改alfie.txt的时候，就会看到如下提示信息：

这是提示已经有了alfie.txt.swp谁的。当看到这个提示信息时，就要考虑一下，是不是自己之前不小心没保存就掉线了，还是说有其它人正在编辑这个文件。如果是自己之前没有保存，那么可以选择删除这个隐藏的swp文件，再编辑就没有这个提示了，这是最简单的方式。或者输入 vi -r alfie.txt 会看到自己之前保存的内容，重新 :wq! 保存一下，然后再删除.alfie.txt.swp 文件即可，自己之前的修改也就还在。

如果是其它人在编辑这个文件，你就输入O，表示只读模式打开，E是直接编辑，R是恢复文件，Q是退出不编辑了，A是中止操作。其实还有个D指令，直接删除隐藏文件。
