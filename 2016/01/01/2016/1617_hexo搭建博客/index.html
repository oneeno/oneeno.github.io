Hexo搭建博客自己

tags: 教程

第一章 前期准备

1，安装前所需环境介绍

安装 Hexo 之前，需要确保 PC 中已经安装以下工具：

  • Node js 地址：https://nodejs.org
  • Git 地址：https://git-scm.com/download/win

2，安装 Git

官方下载地址：https://git-scm.com/download/win

下载后双击安装，直接下一步，安装即可，安装时，需要勾选 Add to PATH 选项

3， 安装 Node js

官方下载地址：https://nodejs.org

打开下载地址后，选择长期支持版下载

下载后，双击安装，然后一直点下一步至安装完成。

第二章 安装 Hexo

1, 安装 Hexo

  • 命令：npm install -g hexo-cli

​ 说明：-g 表示全局安装， hexo-cli 为所安装的包

  • 安装后验证：在 cmd 中输入命令 hexo -v 可查看 hexo 版本

至此，已经成功安装了Hexo,接下来配置Github

注意：建议安装最新版本的 Hexo， 以及 推荐的 Node js

第三章 配置 Github

如果没有 Github 账户，需要先注册一个 Github 账户。

1， 在 Github 上创建仓库

  • 新建一个名为 username.github.io 的仓库（username为 Github 账户的用户名 ）

  • 比如，如果你的 github 用户名是 test，那么就新建名为 test.github.io 的仓库（必须是用户名，其它名称都无效），将来访问网站的地址就是 https://test.github.io 了。由此可见，每个 github 账户最多只能创建一个这校招可以直接使用域名访问的仓库。

注意事项：

  • 注册的邮箱一定要验证，否则不会成功
  • 仓库名字必须是 username.github.io，其中 username 是 Github 的用户名
  • 仓库创建成功不会立即生效，需要过一段时间，大概10-30分钟，或者更久
