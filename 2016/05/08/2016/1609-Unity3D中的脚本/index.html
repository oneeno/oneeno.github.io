Unity3D中的脚本

Input类

获取用户事件
在当前这一帧中，如果用户按下 W 应会返回 true,否则就会返回 false

bool b = Input.GetKeyDown (KeyCode.W);
if (b) {
    print ("往前走");
}

Input.GetKeyDown() 用来检测按键按下的事件

if(Input.GetKeyDown(KeyCode.S))
{
    print 只能在 MonoBehaviour 的子类中使用，其他情况下只能使用 Debug.log() 打印
    Debug.log("往后退");
}

Input.GetKeyUp() 用来检测按键弹起的事件

if(Input.GetKeyUp(KeyCode.F))
{
    print("弹起了1键");
}

Input.GetKey() 用来检测按键持续按住的事件

if(Input.GetKey(KeyCode.F))
{
    print("按下了f键");
}
