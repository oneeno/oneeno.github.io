零基础如何系统地学习前端开发

基础知识
1、HTML + CSS
这部分建议在W3school在线教程上学习，边学边练，每章后面有小测试。
然后模仿一些网站做些页面。
在实践中积累了一些经验后，可以系统的读一两本书。《Head First HTML 与 CSS 中文版》
2、Javascript
先在W3school上学习。然后看《Javascript语言精粹》
有了以上基础后，可以做一般的静态网页设计。
进阶
1、CSS。
必看《精通CSS》，看完后，应该对盒子模型，流动，Block，inline，层叠，样式优先级等概念非常了解。
作为练习，可以看下《CSS艺门之匠》。它对标题，背景，圆角，导航条，table，表单等主题都有详细的介绍。

2、Javascript
有了以上基础后，需进一步学习JS
a> 框架
推荐JQuery,简单易用，在W3school简单学习JS后，直接上手JQuery即可完成一些简单的项目。
学习方法：照着产品文档做几个页面就行了，不用面面俱到，以后遇到问题查文档就行。
b> Javascript 语言范式。
c> Javascript 语言内部机制。
JS中变量的作用域，变量传递方式，函数的定义环境与执行环境，闭包，函数的四种调用方式（一般函数，对象的方法，apply,call），以及四种调用方式下，’this’指向的是谁。这部分内容在《Javascript语言精粹》中详细了解。另外，必须json。
d> dom编程
这是Web前端工程师的核心技能之一。必读《Dom编程艺术》，另外《高性能Javascript》里面对Dom编程的部分讲得也很好。
e> Ajax编程
这是另一个核心技术。在网上查些资料，了解来龙去脉，百度百科，维基百科上看下就行。
f> 了解浏览器的差异性。
浏览器差异内容很多，在实践中多多积累。
对于浏览器的渲染模式，DOCTYPE等内容应该系统学习。
3、HTML5和CSS3.
代码层面
1> 易维护
对于页面必须理解“样式“、”数据“、”行为“三者分离，对应的就是CSS，HTML，JS。
对于JS代码，最好了解设计模式，重构，MVC等内容。
2> 可测性。
3> 高性能。
必读《高性能Javascript》
4> 低流量。
工程层面
代码管理，必须学会使用SVN和Git。
其次是代码的构建。需要进行依赖管理、模块合并、各种编译。必须要学会使用Grunt、Gulp等前端构建工具。
其它
1、交互设计
推荐看《简约至上》
2、后端
必须了解一门后端语言。入手难度比较低的是PHP。
Nodejs，推荐使用Express框架做一些后端服务的开发
3、Android和iOS开发
至少要了解Native开发的基本流程。必须了解界面构建的技术。
