沟通的迷思

澄清人们对沟通的迷思，与说明沟通的原则同等重要。为了避免陷入人际的困扰，请谨记不要犯以下的沟通迷思。

  1. 沟通得越多不见得沟通得越好
    不足会产生问题，然而，沟通过头也会制造问题。有时候过度的沟通只是平白浪费时间。说太多也可能使问题更加恶化。太多的负面沟通只会导致更多的负面结果。
  2. 意思不在字眼里
    我们常犯的最大错误，便是认定沟通就是把话说出来。然后这并不能保证信息接收者所接收到的意思永远符合传送者所想传达的意思。比如：你说了一句具有深刻意义的话，可能别人完全有不同的解释。
  3. 成功的沟通并不表示彼此了解
    双方彼此了解的时候是成功沟通的指标，但有时候沟通得很成功的两个人并非互相了解。满意的关系有时候依赖片面的了解。
  4. 人的反应并非只针对单一事件或特定的对象
    千万不要认为结果都是单一事件所造成的。有很多情境下，有许多因素都会影响他人如何回应你。由于沟通是种交流、持续不断与共同合作的过程，因此任何事件都不会是真空孤立、单独存在的。
  5. 沟通不会解决所有的问题
    有时候，即使天时地利人和，一切都计算得很完美，仍然没有办法解决问题。
