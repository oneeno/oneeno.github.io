人际沟通的本质

部分学者认为：唯有沟通的品质才能取代数量用以作为划分人际沟通类型的标准。品质的定义，主张沟通只有在一方将另一方视为独一无二的个体，且无视沟通发生的场合或参与者的人数时才成立。当我们以品质的定义作为标准时，人际沟通的反而便成了非个人化的沟通。品质沟通的特征如下

  1. 独特性。
    在低个人化的沟通场合，主要依据社会规范作为沟通的准则。品质性的沟通关系则讲求独特的规范与社会角色。
  2. 不可替代性
    人际沟通的独特性使其对每段关系来说都是无可取代的。这项特征告诉我们，为何当一段亲密关系或感情由浓转淡时我们会如此难过，因为我们都知道它是不可取代的，无论身边有现金的朋友、亲人，都没有一段关系能给予我们像那样的感受。
  3. 相互依存性
    简单的讲，就是你和对方在一段关系中是命运相连的。有品质的人际观点中，某些相互依存的关系也可能 超越相互联结的命运。在人际关系中，我们的认同根植于我们与别人互动的本质。比如：当我们说某人具有吸引力时，是因为有人受他吸引；某人是领导者，是因为有人愿受他领导；同样地，某人可爱，是因为有人欣赏他。
  4. 它通常（但不总是）包含着自我揭露
    在非个性化关系中，我们通常不会过度表现自己，但是在个人化关系中，我们比较容易自在地分享感情和想法。
  5. 内存的奖赏
    在非个人化关系中，我们并不会对在场的人有所付出。为了达到某种目的，我们会坐在教室中听课，或和想要买我们的二手手机买主简短交谈，但是我们并不会去经营跟他们的个人化关系。与此相反，在亲密关系中，我们会花时间和朋友、爱人等相处，因为这能满足我们的内存。对我们来说，谈论什么都无所谓，最重要的是，我们彼此对这段关系的重视。
