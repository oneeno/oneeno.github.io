数据库操作

如果关闭了 mariadb 服务，再输入 mysql 就会报错

mysql 登录进去，就可以进行数据库操作了。

数据库中有三个基本概念。

  • 库：存数据表的仓库，可以理解为一个网站一个仓库。他对应着硬盘上的一个文件夹，每个库都有自己的名字。
  • 表：和 excel 表一样的表现形式来存储数据，二维表，他对应着文件夹中的一个文件，其实一个表需要多个文件来存储数据，每个表都有自己的名字。
  • 记录：表的第一行叫做表头，也叫做列名，剩下的行都是实际的数据。一行记录是一条数据，多数都会包含一个对象的全部描述信息。
